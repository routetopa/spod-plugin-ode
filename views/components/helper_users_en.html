Members helper
Members page permits active or registered user visualization/exploration and several types of search on SPOD platform. There are three sections to explore the users including: All, On line and Search.
1.
All
“All” page shows all users who signed up on the SPOD platform.
2.
Online
”On line” page shows only users online on SPOD platform.
3.
Search
”Search” page allows search of users by name or gender.
