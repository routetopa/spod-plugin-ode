What's New helper
What's new page shows all the activities (e.g. posts, likes, photos etc...) to everybody chronologically ordered even for users that are not my friends and it provides a switch between the activities of all the users and only the friends in the network, clicking on button, named Show only my friends' activities.
1
Post an OpenData visualization
By Datalet button (circled in Figure.) near the post creation control, you can create new posts attaching an OpenData visualization(Datalet).
2
Create a visualization
Everyone can, without technical expertise, create and share visualisation of a chosen dataset with few simply steps using the wizard . The wizard assist the user step-by-step through the creation (or modification) of a datalet. The first step is the dataset selection (to not to be confused with datalet) from a compatible provider. There are three methods the user can choose a dataset:
  • It can be selected from a searchable drop-down list of datasets available from one of the associated data providers.
  • It can be selected while exploring interactively the datasets available from associated data providers using a treemap.
  • It can be manually entered as a URL.
3
Fields selection
Once the user chose a dataset, he proceed to the second step: fields selection. The controllet downloads the dataset in a compatible format (for instance: JSON), parses it and builds a treeview (a hierarchical list) of fields found in the dataset. The user can select the fields he wants to include into the datalet and then proceeds to the last step. A preview of the data related to the selected fields will be shown on the table on the right side.
4
Select visualization
In the final step the user has to setup the chart. The Wizard obtains the list of charts, also including name, type and description of inputs and settings supported by each chart. The user must select, for each input required by the chart, a corresponding field from the set he selected in step two. Optionally, the user can customize the chart by changing the settings (e.g. he can set a title and a comment for the graph, put labels on the axes, and so on). While the user makes changes to the inputs or the settings, the Wizard updates the chart preview. When the user is satisfied with the look of the chart, he clicks the "Add" button.
5
Post visualization in a comment
The same feature is available when the user replies with a comment.
6
Use MySpace contents
An user can post a Datalet previously created in the MySpace by using the related button.
